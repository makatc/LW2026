# Stack Oficial
- **Backend:** Django 5.x (Python)
- **Base de Datos:** PostgreSQL + pgvector
- **Frontend:** Django Templates + Tailwind CSS (via CDN o build)
- **Iconos:** FontAwesome (fas fa-*)
- **IA:** Google Gemini 2.0 Flash

## Reglas de Codigo UI
- No usar CSS inline (`style="..."`). Usar clases de Tailwind.
- Las imagenes deben tener `alt` descriptivo.
- El logo debe cargar desde `{% static 'img/logo_legalwatch.png' %}`.
